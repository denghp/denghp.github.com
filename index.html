A Bash script for App Store auto-ingest Tool

Written by nianliang on Aug. 28, 2011 .

A couple of days ago, Apple released the auto-ingest tool. Now, with this tool we can easily download sales daily reports.

But, it's obviously, there are two problems.

  1. Enter the password on the command line is not safe
  2. Inputting the command every time is boring, as it is too long

Here is a very draft Bash script to solve the problems, it will prompt for a password when script running.

#!/usr/bin/env bash

# Usage:
# ./daily-report
# ./daily-report -d 20110909
#
# Copyright 2011, Mo Nianliang
# Released under the MIT license

#---------------------------------------------------------------------
# Configure your profile here:
#---------------------------------------------------------------------
# Apple ID
username=<your_apple_id>
vendorid=<your_vendor_id>
# get ...

Bash: Find And Delete Files Or Directory Recursively

Written by nianliang on July 14, 2011 .

Problem

Just want to delete some certain files from a directory and its sub-directorys, e.g. all temporary files, or 'build' directory of all Xcode projects.

I've tried using the for statement to loop through the results which returned from the find statement.

For temporary files: 

for f in `find myprojects/ -name "*~"`; do
    rm $f
done

Or a directory:

for f in `find myprojects/ -type d -name "build"`; do
    rm -rf $f
done

However, it's broken for some of the directorys which name contains spaces.

Solution

find myprojects/ -name "*~" -exec rm '{}' +

Or

while read f; do rm -rf ...
